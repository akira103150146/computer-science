APCS 實作題怎麼評分：看懂「部分分」

APCS（大學程式設計先修檢測）的實作題不是「全對才給分」。每一題在後台都有多組測資，你的程式讀一筆輸入、印一筆輸出，系統逐字元比對你的輸出與標準答案，每通過一筆測資就拿到對應的分數——這就是部分分（partial credit）。

更進一步，許多題目把測資分成幾個子任務（subtask），每個子任務有自己的資料範圍與配分。例如一題可能這樣設計：

子任務資料範圍配分該寫的解法
子任務 1n ≤ 10030%暴力 O(n²) 就能過
子任務 2n ≤ 500030%稍微優化
子任務 3n ≤ 10⁶40%需要 O(n) 或 O(n log n)

這代表：就算你想不出最佳解，先把暴力法寫對拿下前面的子任務分數，也遠勝於交白卷。考場策略是「先求正確、再求快」：用最直覺、最不會寫錯的方法先 AC 小範圍，行有餘力再優化。本附錄每題的參考解多半直接給「能過全部測資」的版本，但你練習時可以刻意先寫暴力版，體會部分分的拿法。

⚠️
部分分最容易被「格式」吃掉

評測是逐字元比對：多一個空格、少一個換行、行尾多了空白都會讓本來會過的測資判錯。輸出務必照題目範例，尤其注意「最後一個數字後面不要再接空格、整份輸出最後要有一個換行」。本附錄的參考解都用 i==n-1?'\n':' ' 這類寫法處理行尾，值得學起來。

📝 1-T01　進位轉換（任意進位互轉） 基礎

讀入「來源進位 b1、數字字串 x、目標進位 b2」，把 x（b1 進位）轉成 b2 進位輸出。b1、b2 皆為 2、8、10、16 之一；超過 9 的位用大寫 A~F。結果為 0 時輸出 0。

輸入 b1 x b2（b1,b2∈{2,8,10,16}，x 為 b1 進位、0≤值≤10⁵）　輸出 x 的 b2 進位表示

輸入 輸出
10 255 16FF
2 1101 1013
16 FF 211111111
看參考解

C 參考解

#include <stdio.h>
#include <string.h>
long long toDec(const char *s, int base){      /* 任意進位字串 -> 十進位 */
    long long n = 0;
    for (int i = 0; s[i]; i++){
        int d;
        char c = s[i];
        if (c >= '0' && c <= '9')      d = c - '0';
        else if (c >= 'A' && c <= 'F') d = c - 'A' + 10;
        else                           d = c - 'a' + 10;
        n = n * base + d;
    }
    return n;
}
void fromDec(long long n, int base, char *out){ /* 十進位 -> base 進位字串 */
    const char *D = "0123456789ABCDEF";
    char buf[64]; int k = 0;
    if (n == 0){ out[0] = '0'; out[1] = '\0'; return; }
    while (n > 0){ buf[k++] = D[n % base]; n /= base; }
    for (int i = 0; i < k; i++) out[i] = buf[k - 1 - i];
    out[k] = '\0';
}
int main(void){
    int b1, b2; char x[64], out[64];
    scanf("%d %63s %d", &b1, x, &b2);
    fromDec(toDec(x, b1), b2, out);
    printf("%s\n", out);
    return 0;
}

Python 參考解

def to_base(n, base):
    if n == 0:
        return "0"
    digs = "0123456789ABCDEF"
    out = ""
    while n > 0:
        out = digs[n % base] + out
        n //= base
    return out
b1, x, b2 = input().split()
n = int(x, int(b1))          # 任意進位字串 -> 十進位
print(to_base(n, int(b2)))
📝 1-T02　ASCII 與字元位移（凱撒位移） 基礎

讀入位移量 k（可正可負）與一行字串。對每個英文字母做位移：A~Z、a~z 各自在 26 字母內循環（(字母−基底+k) mod 26），其餘字元（數字、空白、標點）原樣保留。利用「字元 = ASCII 整數」的觀念計算。

輸入 第一行 k（−1000≤k≤1000）；第二行一行字串（長度 0~100）　輸出 位移後字串

輸入 輸出
3
ABCxyz		DEFabc
1
Hello, World!		Ifmmp, Xpsme!
看參考解

解題重點與說明

教學重點：Python 的 % 對負數已回正，C 需先用 ((k%26)+26)%26 把位移化到 0~25。

C 參考解

#include <stdio.h>
char shiftCh(char c, int k){
    int m = ((k % 26) + 26) % 26;                 /* 化為 0..25 */
    if (c >= 'A' && c <= 'Z') return (char)('A' + (c - 'A' + m) % 26);
    if (c >= 'a' && c <= 'z') return (char)('a' + (c - 'a' + m) % 26);
    return c;
}
int main(void){
    int k; scanf("%d", &k); getchar();            /* 吃掉換行 */
    int c;
    while ((c = getchar()) != EOF && c != '\n')
        putchar(shiftCh((char)c, k));
    putchar('\n');
    return 0;
}

Python 參考解

def shift_char(c, k):
    if 'A' <= c <= 'Z':
        return chr((ord(c) - ord('A') + k) % 26 + ord('A'))
    if 'a' <= c <= 'z':
        return chr((ord(c) - ord('a') + k) % 26 + ord('a'))
    return c
import sys
data = sys.stdin.read().split("\n")
k = int(data[0])
text = data[1] if len(data) > 1 else ""
print("".join(shift_char(c, k) for c in text))

📝 1-T04　二進位中 1 的個數（popcount） 基礎

讀入非負整數 n，輸出其二進位表示中「1」的個數（人口計數）。

輸入 n（0≤n≤10⁹）　輸出 n 的二進位中 1 的個數

輸入 輸出
133
00
2558
看參考解

解題重點與說明

進階技巧：n & (n-1) 會清掉最低位的 1，反覆做直到 0 也能算出 1 的個數（Brian Kernighan 法）。

C 參考解

#include <stdio.h>
int popcount(long long n){
    int c = 0;
    while (n > 0){ c += (int)(n & 1); n >>= 1; }
    return c;
}
int main(void){ long long n; scanf("%lld", &n); printf("%d\n", popcount(n)); return 0; }

Python 參考解

def popcount(n):
    c = 0
    while n > 0:
        c += n & 1
        n >>= 1
    return c
n = int(input())
print(popcount(n))
📝 1-T05　判斷是否為 2 的次方 進階

讀入正整數 n，若 n 恰好是 2 的某次方（1,2,4,8,16,…）輸出 yes，否則輸出 no。

輸入 n（1≤n≤10⁹）　輸出 yes 或 no

輸入 輸出

1yes
6no

看參考解

解題重點與說明

def is_pow2(n):

n = int(input())
print("yes" if is_pow2(n) else "no")

📝 1-T07　補數（N 位元一補數與二補數） 進階

讀入 N 與非負整數 x（0≤x<2^N）。輸出兩個數（空白分隔）：x 在 N 位元下的一補數（所有位元反轉）與二補數（一補數再加 1，並對 2^N 取模回繞）。

輸入 N x（2≤N≤16，0≤x<2^N）　輸出 一補數 二補數

輸入 輸出
8 5250 251
4 114 15
8 2550 1
看參考解

解題重點與說明

重點：在 N 位元下「反轉」要對 mask = 2^N − 1 做 XOR，否則高位的 1 會殘留；二補數加 1 後要回繞（& mask）。

C 參考解

#include <stdio.h>
int main(void){
    int n; long long x; scanf("%d %lld", &n, &x);
    long long mask = (1LL << n) - 1;
    long long ones = x ^ mask;            /* 一補數：位元反轉（限制在 N 位） */
    long long twos = (ones + 1) & mask;   /* 二補數：加 1 後回繞 */
    printf("%lld %lld\n", ones, twos);
    return 0;
}

Python 參考解

n, x = map(int, input().split())
mask = (1 << n) - 1
ones = x ^ mask          # 一補數：位元反轉
twos = (ones + 1) & mask # 二補數
print(ones, twos)
📝 1-T08　N 位元有號數溢位後的數值 挑戰

把十進位整數 v 存進「N 位元二補數」整數變數後，實際會被解讀成什麼值？合法範圍是 [−2^(N−1), 2^(N−1)−1]，超出範圍會以 2^N 為週期回繞（wrap-around）。請輸出 v 存入後實際代表的有號值。

輸入 N v（2≤N≤16，−10⁶≤v≤10⁶）　輸出 回繞後的有號值

輸入 輸出
8 200-56
8 -1-1
4 8-8
8 127127
看參考解

解題重點與說明

例：8 位元存 200，二補數最大正值為 127；200 的位元樣式 11001000 被解讀為 −56（200 − 256）。

C 參考解

#include <stdio.h>
int main(void){
    int n; long long v; scanf("%d %lld", &n, &v);
    long long m = 1LL << n;
    long long r = ((v % m) + m) % m;   /* 化到 0..m-1（處理負數） */
    if (r >= (m >> 1)) r -= m;          /* 高位為 1 -> 負數 */
    printf("%lld\n", r);
    return 0;
}

Python 參考解

n, v = map(int, input().split())
m = 1 << n
r = v % m            # Python % 對負數自動回正：0..m-1
if r >= (m >> 1):    # 最高位為 1 -> 視為負數
    r -= m
print(r)
📝 1-T09　row-major 二維陣列記憶體位址計算 挑戰

給二維陣列的列數 rows、行數 cols、每個元素位元組大小 size、起始位址 base，以及索引 (i, j)（皆從 0 起算）。在 row-major（列優先） 配置下，元素 A[i][j] 的位址為：base + (i*cols + j) * size。請輸出該位址（十進位）。

輸入 rows cols size base i j（1≤rows,cols≤1000；size∈{1,2,4,8}；0≤base≤10⁶；0≤i<rows；0≤j<cols）　輸出 A[i][j] 的位址

輸入 輸出

5 5 1 0 0 00
2 3 8 2000 1 22040

    return 0;
}

Python 參考解

rows, cols, size, base, i, j = map(int, input().split())
print(base + (i * cols + j) * size)

解題重點與說明

思路：對每一列先找最大值的位置 jc，再檢查它是不是該行（第 jc 行）的最小值。

C 參考解

#include <stdio.h>
int a[105][105];
int main(void){
    int M,N; scanf("%d %d",&M,&N);
    for(int i=0;i<M;i++) for(int j=0;j<N;j++) scanf("%d",&a[i][j]);
    for(int i=0;i<M;i++){
        int jc=0;
        for(int j=1;j<N;j++) if(a[i][j]>a[i][jc]) jc=j;   // 該列最大值的行

        for(int r=0;r<M;r++) if(a[r][jc]<a[i][jc]) ok=0;  // 是否為該行最小

    }
    printf("-1\n");
    return 0;
}

    int R,C; scanf("%d %d",&R,&C);

解題重點與說明

思路：掃描每個尚未走訪的 1，用 DFS／BFS 把整座島標記掉，計數 +1。方向陣列上右下左是本章核心技巧。

C 參考解

#include <stdio.h>
int g[105][105], M, N;
void dfs(int i,int j){
    if(i<0||i>=M||j<0||j>=N||g[i][j]!=1) return;
    g[i][j]=2;                      // 標記為已走訪
    dfs(i-1,j); dfs(i+1,j); dfs(i,j-1); dfs(i,j+1);
}
int main(void){
    scanf("%d %d",&M,&N);
    for(int i=0;i<M;i++) for(int j=0;j<N;j++) scanf("%d",&g[i][j]);
    int cnt=0;
    for(int i=0;i<M;i++) for(int j=0;j<N;j++)
        if(g[i][j]==1){ dfs(i,j); cnt++; }
    printf("%d\n", cnt);
    return 0;
}

Python 參考解

import sys
from collections import deque
d = sys.stdin.read().split(); idx = 0
M = int(d[idx]); N = int(d[idx+1]); idx += 2
g = [[int(d[idx+i*N+j]) for j in range(N)] for i in range(M)]
cnt = 0
DIR = ((-1,0),(1,0),(0,-1),(0,1))
for si in range(M):
    for sj in range(N):
        if g[si][sj] == 1:
            cnt += 1; g[si][sj] = 2; q = deque([(si,sj)])
            while q:
                i, j = q.popleft()
                for di, dj in DIR:
                    ni, nj = i+di, j+dj
                    if 0 <= ni < M and 0 <= nj < N and g[ni][nj] == 1:
                        g[ni][nj] = 2; q.append((ni,nj))
print(cnt)
